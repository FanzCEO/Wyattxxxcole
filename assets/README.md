# Assets Directory

This directory contains media assets for the Wyatt XXX Cole website.

## Structure

```
assets/
├── images/          # Photos, headshots, portfolio images
│   ├── hero/        # Hero section backgrounds
│   ├── portfolio/   # Portfolio gallery images
│   ├── shop/        # Product images
│   └── social/      # Social media assets
├── video/           # Video files
│   └── reels/       # Video reels and teasers
└── pdf/             # Downloadable documents
    └── EPK/         # Electronic Press Kit files
```

## Image Guidelines

### Recommended Sizes
- Hero background: 1920x1080px minimum
- Portfolio images: 800x1200px (3:4 ratio)
- Product images: 600x600px (1:1 ratio)
- Avatar/Profile: 400x400px

### Formats
- Photos: JPG (optimized) or WebP
- Graphics with transparency: PNG
- Video: MP4 (H.264)

## Brand Colors for Graphics
- Background Dark: #05060A
- Electric Purple: #9B1DFF
- Neon Cyan: #00FAFF
- Toxic Green: #39FF14
- Chrome/Silver: #C0C4CC
