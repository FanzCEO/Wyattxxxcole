# WYATT XXX COLE - Neon Cyber Rebel

> Neon. Leather. Trouble. The cyber rebel of adult entertainment.

A complete brand website for Wyatt XXX Cole (WXXXC), featuring a cyberpunk aesthetic with neon accents and a professional digital presence.

## Brand Identity

**Name:** Wyatt XXX Cole
**Monogram:** WXXXC

### Brand Pillars
1. **Professional** – Studios can trust reliability on set
2. **Rebel** – Distinctive look and attitude
3. **Futuristic** – Tech-aware, social-savvy
4. **Magnetic** – Fans follow the world, not just the content

## Pages Overview

### Home (`index.html`)
- Hero section with video background support
- Brand introduction
- Quick stats (location, travel status, etc.)
- Featured work preview
- Upcoming cities

### For Studios (`studios.html`)
- Professional bio
- Physical stats & specs table
- Performance categories
- Experience highlights
- Visual showcase (6-12 curated images)
- Booking inquiry form
- Press kit download

### Portfolio (`portfolio.html`)
- Tabbed interface:
  - Photos gallery
  - Video reel
  - Themed/character shoots
  - Behind the scenes

### Schedule (`schedule.html`)
- Interactive calendar
- Upcoming cities list with availability type
- Email notification signup

### Community (`community.html`)
- Social feed with posts
- Pinned announcements
- Polls and interactions
- Subscription CTAs

### Shop (`shop.html`)
- Product categories (Apparel, Prints, Digital, Limited)
- Product cards with hover effects
- Category filter tabs

### Contact (`contact.html`)
- Professional booking form
- Creator collaboration form
- Boundaries statement
- Direct contact info

### Link Hub (`links.html`)
- Minimal Linktree-style page
- Content platform links
- Social media links
- Site navigation shortcuts

### EPK Print Version (`epk.html`)
- Print-optimized layout
- One-page bio, stats, photos
- Save as PDF functionality

## Getting Started

1. Clone the repository
2. Open `index.html` in a browser
3. Replace placeholder content with actual information
4. Add images to `assets/images/`
5. Update social/platform links

### Customization

Edit `css/styles.css` CSS variables to adjust:
- Brand colors
- Typography
- Spacing
- Animation timing
