## Features

- **Responsive Design** – Mobile-first approach
- **Dark Theme** – Warm country aesthetic with earthy tones
- **Animations** – Scroll animations, particle effects, warm glows
- **Interactive Components** – Tabs, forms, calendar
- **Accessibility** – Semantic HTML, ARIA labels
- **SEO Ready** – Meta tags, Open Graph
- **Backend API** – MariaDB support for dynamic content

## Getting Started

1. Clone the repository
2. Open `index.html` in a browser
3. Replace placeholder content with actual information
4. Add images to `assets/images/`
5. Update social/platform links

### Customization

Edit `css/styles.css` CSS variables to adjust:
- Brand colors
- Typography
- Spacing
- Animation timing
